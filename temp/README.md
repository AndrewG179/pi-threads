# Temp Archive

This directory contains archived working notes, review notes, and live-drive artifacts.

Rules:
- non-normative only
- not authoritative for current behavior
- canonical behavior lives in `docs/subagents.md`
- bug history and fix evidence live in `bug-reports/`

The `archive/` subdirectory is for plain-text notes/reviews. Other dated subdirectories may contain captured live-drive artifacts and investigation dumps.
